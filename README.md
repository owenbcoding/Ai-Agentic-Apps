# Agentic AI Applications Notes

This repository contains structured notes for an 8-week learning journey focused on **Agentic AI applications**.

It is organized as a practical study companion that moves from AI/ML foundations to building and operating production-ready agent workflows.

## What This Repo Covers

- Core AI and ML fundamentals (AI vs ML vs Generative AI, classification, regression)
- Deep learning basics (neural networks, CNNs, backpropagation)
- NLP and transformer foundations
- Prompt engineering and RAG concepts
- Agentic AI design patterns (tools, memory, ReAct, planning vs execution)
- No-code agent automation with n8n
- Real workflow integrations (Google Sheets, Slack, Email, webhooks, APIs)
- Production concerns (security, monitoring, deployment choices)
- Career prep (resume and interview preparation)

## Repository Structure

The repository is split by week:

- `Week 1` to `Week 4`: Foundations (AI/ML, deep learning, NLP, LLM basics)
- `Week 5` to `Week 7`: Agent systems and workflow automation
- `Week 8`: Production readiness and interview preparation

Each markdown file represents one lesson/topic for that week.

## Start Here

Open `start here.md` first.  
It acts as the roadmap and checklist for all topics across the 8 weeks.

## How To Use These Notes

1. Follow topics week by week using `start here.md`.
2. Fill each lesson file under:
   - `## Notes`
   - `## Key Points`
   - `## Examples`
   - `## Questions`
3. Mark progress in `start here.md` by checking off completed topics.
4. Revisit Week 8 topics to prepare for project delivery and interviews.

## Who This Is For

- Beginners entering AI/GenAI
- Learners transitioning into Agentic AI roles
- Builders who want a clear, project-oriented study path

## Goal

Build enough conceptual depth and practical workflow experience to design, implement, and explain real-world agentic AI systems with confidence.
