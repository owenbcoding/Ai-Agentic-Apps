## Examples

### AI vs ML vs Generative AI - Quick Examples

- AI: A rule-based customer support bot that routes users to the correct help page.
- ML: A churn prediction model that classifies which customers are likely to leave.
- Generative AI: A GPT-based assistant that writes outreach emails, summaries, or reports.
- Agentic AI: A multi-step agent that reads a task, searches data, uses tools/APIs, and returns a final output automatically.

### Project Ideas from the Course

- Project 1: Agentic Knowledge Base Assistant (RAG System)
   Build a RAG-powered chatbot that answers questions from custom documents by combining intelligent retrieval with generative AI. Showcase skills in Retrieval-Augmented Generation, context-aware responses, and applied NLP.
- Project 2: Smart Personal AI Agent (Multi-Tool Assistant)
   Create a multi-tool AI assistant that connects with external APIs (weather, travel, productivity tools) to perform tasks and deliver personalized responses. Showcase skills in agent loops, tool calling, and AI-driven decision making.
- Project 3: Multi-Agent Research & Task Automation System
    Develop a multi-agent AI workflow where specialized agents collaborate to complete complex tasks. A Planner Agent interprets requests, a Research Agent gathers information, and an Executor Agent generates reports and triggers automations.
- Project 4: AI-Driven Regional Sales Analysis
    Analyze regional sales data with AI to identify trends, forecast outcomes, and suggest growth opportunities for different territories.
- Project 5: AI-Driven Cryptocurrency Dashboard
    Build an interactive dashboard that uses AI insights to interpret real-time crypto market movement, detect patterns, and summarize signals.
- Project 6: Customer Churn Prediction System
    Create a predictive ML pipeline to identify at-risk customers and recommend retention actions based on behavior and engagement data.
