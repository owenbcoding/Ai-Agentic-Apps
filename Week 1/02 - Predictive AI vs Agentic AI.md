## Key Points

- Predictive AI is mostly about forecasting/classification (what is likely to happen next).
- Agentic AI is about autonomous execution (deciding steps, using tools, and completing tasks).
- Predictive systems usually output a score/label/forecast; agentic systems output actions/results.
- Predictive AI can be one component inside an agentic system (agents can call predictive models).
- For portfolio strength, combine both:
  - Predictive model for insight.
  - Agentic layer for action and automation.

## Examples

- Predictive AI example: Customer Churn Prediction System that flags at-risk users and suggests retention actions.
- Predictive AI example: Regional Sales Analysis that identifies trends and likely high-growth regions.
- Agentic AI example: Agentic Knowledge Base Assistant (RAG) that retrieves context and answers user queries.
- Agentic AI example: Smart Personal AI Agent (multi-tool assistant) that calls external tools/APIs to finish tasks.
- Agentic AI example: Multi-Agent Research and Task Automation System with planner, researcher, and executor agents.
- Combined example: A churn predictor + agent workflow that automatically drafts retention outreach and creates follow-up tasks.
